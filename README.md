# weatherDashboardJD
[![License: ISC](https://img.shields.io/badge/License-ISC-blue.svg)](https://opensource.org/licenses/ISC)

## Description
With the app, you can search a certain city within the US. You'll also get the forecast of the next 5 days upon searching. Search history also will persist due to local storage.

## Usage
- Naviagte to the site and from there you'll be able to search a certain city.
